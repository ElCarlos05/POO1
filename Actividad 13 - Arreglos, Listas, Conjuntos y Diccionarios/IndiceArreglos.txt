ÍNDICE DEL PROGRAMA: CLASE ARREGLO CON MENÚ INTERACTIVO PARA LISTAS

1. Definición de la clase Arreglo
   - Atributos:
     • lista: Lista vacía que almacenará valores numéricos.
     • seleccion: Variable para almacenar la opción seleccionada en el menú.
     • terminar: Booleano para controlar la finalización del programa.

2. Métodos principales:
   - separador(self):
     • Imprime una línea divisoria en consola.

   - menu(self):
     • Muestra las opciones del menú:
       1 - Insertar
       2 - Eliminar
       3 - Modificar
       4 - Mostrar
       5 - Salir
     • Recibe la selección del usuario.
     • Valida que la opción esté entre 1 y 5.
     • Si es 5, cambia `terminar` a True.

   - escoger(self):
     • Ejecuta un ciclo mientras `terminar` sea False.
     • Según la opción elegida, llama a los métodos correspondientes:
       • insertar(), eliminar(), modificar(), mostrarLista().
     • Después de cada acción, vuelve a mostrar el menú.

3. Métodos de manipulación de la lista:
   - insertar(self):
     • Solicita al usuario un número entero.
     • Agrega el número a la lista.

   - eliminar(self):
     • Muestra la lista.
     • Solicita la posición del elemento a eliminar.
     • Elimina el elemento en esa posición (ajustado por índice -1).

   - modificar(self):
     • Muestra la lista.
     • Solicita la posición del elemento a modificar.
     • Solicita el nuevo valor para esa posición.
     • Actualiza el valor en la lista.

   - mostrarLista(self):
     • Imprime la lista actual.

4. Ejecución principal
   - Se crea el objeto `datos = Arreglo()`.
   - Se llama a `datos.menu()` para mostrar el menú inicialmente.
   - Se llama a `datos.escoger()` para comenzar el ciclo de interacción.

5. Observaciones
   - Programa interactivo para gestionar una lista.
   - Uso de ciclos y control de flujo para validación de opciones.
   - Fácil de extender con más funcionalidades.
   - Manejo simple de listas y entradas del usuario.
