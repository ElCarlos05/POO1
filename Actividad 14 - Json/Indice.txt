ÍNDICE DEL PROGRAMA: MANEJO DE JSON Y CONSUMO DE API EN PYTHON

1. Importación de librerías
   - json: Para manejar archivos JSON
   - requests: Para realizar solicitudes HTTP

2. Clase GestorJson
   - __init__(self, arch): Inicializa con el nombre de archivo JSON
   - leerjson(self):
       • Lee y carga contenido JSON desde archivo
       • Maneja errores: archivo no encontrado y JSON inválido
   - escribir(self, datos):
       • Escribe datos en formato JSON al archivo con indentación
   - modificarJson(self, llave, valor):
       • Modifica un valor específico en el JSON
       • Llama a leerjson para obtener datos actuales y luego escribe cambios
   - obtenerPokemon(self, pokemon):
       • Consulta API pública https://pokeapi.co para obtener datos de un Pokémon
       • Guarda la respuesta JSON en archivo local
       • Maneja errores HTTP y de solicitud

3. Uso del programa
   - Instancia GestorJson con archivo "pokemon.json"
   - Llama obtenerPokemon con nombre "pikachu"
   - Lee y muestra en consola la información descargada

4. Observaciones
   - Manejo básico de excepciones para robustez
   - El método modificarJson tiene error tipográfico (leer_json → leerjson)
   - Uso de indentación en JSON para legibilidad
