ÍNDICE DEL PROGRAMA: FLASK CON RESTRICCIÓN DE IP

1. Importaciones
   - Flask: framework web
   - request: para acceder a la solicitud HTTP
   - abort: para devolver errores HTTP

2. Configuración
   - app = Flask(__name__): instancia la aplicación Flask
   - ALLOWED_IP = '10.2.82.25': IP permitida para acceder

3. Middleware antes de cada solicitud
   - @app.before_request: función que se ejecuta antes de cada ruta
   - limit_remote_addr(): verifica si la IP remota coincide con ALLOWED_IP
       • Si no coincide, aborta con código HTTP 403 (Forbidden)

4. Ruta principal
   - @app.route('/'): ruta raíz
   - hello_world(): responde con "Hola Mundo"

5. Ejecución
   - app.run(host='0.0.0.0', port=5000): inicia servidor en todas las interfaces y puerto 5000

6. Observaciones
   - Restricción de acceso sencilla basada en IP
   - Abort con 403 bloquea a clientes no autorizados
