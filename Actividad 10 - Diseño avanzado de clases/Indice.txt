ÍNDICE DEL PROGRAMA: CLASE PISTO CON ENCAPSULAMIENTO Y MÉTODOS

1. Definición de la clase pisto
   - Atributos:
     • __hielera (privado): Almacena un valor inicial (string).
     • hielera2 (público): Otro atributo público.
   
2. Métodos
   - pistear(self):
     • Modifica el atributo privado __hielera.
     • Retorna el nuevo valor: "im going back to 505".

   - pistear2(self):
     • Modifica el atributo público hielera2.
     • Retorna el nuevo valor: "nah, esta vacia boludo".

3. Creación de objeto y pruebas
   - fiesta = pisto("Carton en hielera", "Unas cuantas frias")
   - Llamadas a métodos:
     • print(fiesta.pistear())   → imprime: im going back to 505
     • print(fiesta.pistear2())  → imprime: nah, esta vacia boludo

4. Observaciones
   - Uso de atributo privado con doble guion bajo (__hielera) para encapsulamiento.
   - El método pistear cambia el valor interno privado.
   - El método pistear2 cambia el valor público.
   - Muestra cómo manejar atributos privados y públicos dentro de una clase.
